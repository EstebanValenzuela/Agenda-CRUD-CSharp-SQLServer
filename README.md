# Agenda CRUD - C# Windows Forms

Sistema de agenda electrónica desarrollado en C# utilizando Windows Forms y SQL Server.

##  Funcionalidades

- Insertar contactos
- Buscar contactos
- Modificar contactos
- Eliminar contactos
- Validación de correo electrónico
- Validación de teléfonos con formato

##  Tecnologías utilizadas

- C#
- Windows Forms
- SQL Server
- ADO.NET

##  Arquitectura

El proyecto está desarrollado utilizando arquitectura por capas:

- Capa de Presentación
- Capa de Datos
- Capa DAO
